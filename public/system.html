Gestión del servicio

Estas operaciones requieren que el proceso corra con privilegios de Administrador de Windows.

Instalar servicio
Registra KLSyncBridge como servicio Windows con arranque automático y recovery.
Desinstalar servicio
Detiene y elimina el servicio Windows. La configuración y datos no se borran.
Reinstalar servicio
Desinstala e instala en un solo paso. Útil para aplicar cambios de configuración.

Acceso directo

Recrea el acceso directo en el Escritorio y en el Menú Inicio apuntando a http://localhost:3847. Útil si se eliminó el acceso directo accidentalmente.

Reinicialización del sistema

Ejecuta el setup nuevamente. Solo genera los componentes que falten (clave de cifrado, base de datos, JWT secret). No borra datos ni resetea el usuario admin si ya existe.
